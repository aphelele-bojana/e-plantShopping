# Paradise Nursery 🌿

Paradise Nursery is a simple React + Redux plant shop application where users can browse plants, add them to a cart, and manage quantities.

## Features
- Landing page with company name and Get Started button
- Product listing page with plants
- Add to cart functionality
- Shopping cart with quantity update and remove options
- Redux Toolkit state management

## Tech Stack
- React
- Redux Toolkit
- JavaScript
- CSS
